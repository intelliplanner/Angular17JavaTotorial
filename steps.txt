Comipler Usage
- Start your NodeJS server (node server.js).
- Run Angular (ng serve).
- Open http://localhost:4200, type Java code, and click Compile & Run.
- The NodeJS server will invoke your local JDK compiler at D:\Software_Installed_Dir\java\jdk1.8.0_192\bin.
